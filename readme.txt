OpenID Connect Client Extension for phpBB

version : see composer.json

phpBB uses the Lusitanian/PHPoAuthLib library. 
The OAuthSD Service for Lusitanian/PHPoAuthLib is included in 
.../oauthsd-phpbb/OAuth/OAuth2/Service/Oauthsd.php 
and should be copied to 
.../vendor/lusitanian/oauth/src/OAuth/OAuth2/Service/ folder.

See: 
https://area51.phpbb.com/docs/dev/3.2.x/extensions/tutorial_authentication.html
https://oa.dnc.global/web/110
https://github.com/bdegoy/oauthsd-phpbb
